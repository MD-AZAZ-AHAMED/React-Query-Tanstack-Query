TanStack Query Demo
A simple React app demonstrating TanStack Query for efficient data fetching, caching, and synchronization.
Features

Fetch products from DummyJSON API.
Automatic loading states and error handling.
Infinite queries for product lists.
Optimistic updates on mutations.

Tech Stack

React 18
TanStack Query v5
React Router v6
Tailwind CSS

Installation

Clone the repo: git clone <your-repo-url>
Install dependencies: npm install
Run dev server: npm run dev

Usage
Navigate to /products for the list or /product/:id for details. Data fetches automatically!
